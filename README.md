# Event Reservation App

Este projeto contém um sistema simples de gerenciamento de reservas de salas (desafio técnico).

## Estrutura
- **backend/** → API REST (Node.js + Express + Sequelize)
- **frontend/** → Interface web (React + Axios)

## Como rodar
### Backend
```bash
cd backend
cp .env.example .env
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```
